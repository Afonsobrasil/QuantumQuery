# QuantumQuery

* QuantumQuery is an task manager that manages tasks easier.

## Install QuantumQuery

```bash
npm i --save quantumquery
```

## What you can do with QuantumQuery?

* Add Tasks

* Execute Tasks

* Run Quantum Tasks

and more....
